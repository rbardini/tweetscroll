Tweetscroll

Automatically show new tweets on Twitter web interface
  1. Drag Tweetscroll to your bookmarks bar
  2. Go to Twitter.com and click your Tweetscroll bookmark
  3. Done. New tweets will start showing up automatically ;)

Tested on Chrome 12, Firefox 5, IE9, Opera 11 and Safari 5.

See also User script, Chrome extension, Opera 12 extension
